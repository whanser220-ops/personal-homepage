About

从静态网页理解 Web 的入口

我会把这个主页当作一个长期练习项目：每学会一个新知识，就给网站增加一个小功能。 当前版本只包含静态资源，浏览器打开 `index.html` 就能运行。

后续可以逐步加入表单、接口、数据库、登录、后台管理和自动化部署，让它从静态页面演进成完整应用。

Stack

当前学习内容

01

HTML

组织网页结构，理解标题、段落、链接、图片和语义化标签。

02

CSS

控制页面排版、颜色、响应式布局和不同屏幕尺寸下的显示效果。

03

JavaScript

添加少量交互，例如动态年份、导航状态和后续的表单逻辑。

04

部署

用 GitHub 保存代码，用 Nginx 把服务器上的静态文件返回给访问者。
